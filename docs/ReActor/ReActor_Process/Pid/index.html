Module ReActor_Process.Pid

type t = string * string * int

The process identifier type.

It consists of a tuple of the node_name, the scheduler_id, and the process_count at the point of spawning.

This means every given Pid.t is fully unique, but also indicates exactly where it's running and relatively how old it was (lower process_count indicate an older process).
